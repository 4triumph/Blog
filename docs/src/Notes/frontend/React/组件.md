## faker.js

faker.js 是一个 JavaScript 库，用于生成随机数据。它可以用于模拟后端数据，也可以用于前端数据的生成和测试。

地址：[faker.js](https://fakerjs.dev/)

## supabase
supabase 是一个开源的后端服务，提供了实时数据库、身份验证、文件存储等功能。它可以用于构建实时应用，也可以用于构建传统的后端应用。

地址: [supabase](https://supabase.com/)
